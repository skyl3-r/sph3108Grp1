Pseudocode for lectureCode.R

START

1. Install and load the required libraries.

2. Define constants:
   - timesteps = 12
   - beta = 0.1

3. Create a table of cities containing:
   - city name
   - latitude
   - longitude
   - population

4. Extract the longitude and latitude columns as coordinates.

5. Compute pairwise distances between every pair of cities.

6. Label the distance matrix rows and columns with city names.

7. Convert the distance matrix into a long edge table with:
   - from
   - to
   - distance

8. Remove self-loops where a city points to itself.

9. For each edge, assign a travel frequency:
   - if distance < 500 km, assign high travel frequency
   - else if distance < 1000 km, assign medium travel frequency
   - else assign low travel frequency

10. Build a directed graph using:
    - cities as nodes
    - travel connections as edges

11. Set all cities to susceptible (status = 0).

12. Set Beijing as initially infected (status = 1).

13. Create an empty table to store infection status across timesteps.

14. For each timestep from 1 to timesteps:
    - save the current infection status of all cities
    - find all infected cities
    - for each infected city:
      - find its outgoing neighboring cities
      - for each neighboring city:
        - if the neighbor is susceptible:
          - calculate infection probability using:
            beta
            x travel frequency
            x normalized source population
            x normalized target population
          - generate a random number
          - if the random number is less than the infection probability:
            mark the neighboring city as infected

15. Add a timestep column to the saved infection table.

16. Reshape the infection table from wide format to long format.

17. Join city coordinates and population information to the long table.

18. Load world country boundary data.

19. Filter the map data to keep only China.

20. Plot:
    - China boundary
    - city points colored by infection status
    - one facet for each timestep

END
